# 🎵 DesiBeats - Free Desi Music Player

A free, ad-free music player for Indian music - Haryana, Bollywood, Punjabi, Rajasthani & more!

## 🚀 Live Demo

Coming soon on Vercel!

---

## Features

- **Stream Unlimited Music** - Search and play songs from YouTube
- **Genre Browsing** - Haryanvi, Bollywood, Punjabi, Rajasthani, Bhojpuri, Hip Hop, Lo-Fi, Trending, Old Hits, DJ Mix
- **Like Songs** - Save your favorite songs
- **History** - Automatically saved played songs
- **Queue Management** - Add songs to queue, play next
- **Playlists** - Create and manage playlists
- **Lyrics View** - Placeholder for lyrics
- **PWA** - Install as app on mobile
- **Dark Theme** - Spotify-like UI

---

## Tech Stack

- Next.js 14
- React
- TypeScript
- Tailwind CSS
- YouTube API (via CORS proxy)
- Lucide Icons
- PWA Support

---

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open http://localhost:3000
